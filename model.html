User attributes were identified using observation. Both online and direct. Then, an affinity diagram was created to organise the found data, synthesize it into groups and to name those groups. The results were six user types.

Guerrila Personas


Three of those user types were used for the creation of the personas: the animal lover, the doting grandparent, and rich + retired, because they were identified as the loneliest user group. The three user types were then used in combination with an empathy map.
